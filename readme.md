
# Sistema de Gestão Industrial – Transformadores de Alta Potência

## Curso

Técnico em Desenvolvimento de Sistemas

## Tema

Fábrica de Transformadores de Alta Potência e Indústria 4.0

---

## Descrição

Este projeto foi desenvolvido em C# como uma aplicação de console, com o objetivo de simular um sistema de gestão industrial baseado em conceitos da Indústria 4.0.

O sistema permite o cadastro e análise de dados de sensores de transformadores, auxiliando no controle operacional e na tomada de decisões.

---

## Funcionalidades

### Controle de Operação

* Cadastro de máquinas
* Registro de leituras de sensores
* Exibição dos dados cadastrados
* Cálculo da média das leituras

### Controle de Segurança

* Identificação de leituras acima da média
* Identificação de leituras acima de 85°C
* Definição do status do transformador (aprovação ou inspeção)

### Controle de Leitura Crítica

* Pesquisa de temperatura específica
* Verificação de ocorrência
* Exibição das posições encontradas

### Controle Organizacional de Dados

* Ordenação das leituras em ordem crescente
* Ordenação das leituras em ordem decrescente
* Identificação das 3 maiores temperaturas
